# TAREA DE FLUJO NO. 1
"Crear un programa en C# que lea el sueldo de un empleado, calcule el ISR (Impuesto Sobre la Renta) y muestre en pantalla el resultado, de acuerdo con una escala salarial disponible en la plataforma de la DGII."


# FUNCIONAMIENTO:
Este programa funciona como una herramienta interactiva para el cálculo de impuestos sobre la renta (ISR) y deducciones de nómina, con las siguientes características principales:  

- Entrada de Datos: Solicita el ingreso de un sueldo bruto inicial para realizar todas las proyecciones.  

- Procesamiento: Utiliza la clase CalculadoraISR para calcular los descuentos de AFP (2.87%), SFS (3.04%) y determinar el monto de ISR mensual basado en escalas impositivas anuales.  

- Interacción y Reporte: Ofrece un menú de opciones para consultar de forma aislada cada deducción o visualizar un resumen completo de nómina que detalla el sueldo neto final.
